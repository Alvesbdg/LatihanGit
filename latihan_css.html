Latihan CSS

Ini adalah contoh penggunaan CSS.

CSS adalah bahasa desain yang memungkinkan Anda mengatur tampilan dan tata letak elemen-elemen HTML pada halaman web

Dengan CSS, Anda dapat mengubah warna, font, spacing, dan bahkan membuat tata letak yang responsif untuk berbagai perangkat

Cascading Style Sheets, adalah bahasa yang digunakan untuk mengatur tampilan elemen HTML. Dengan CSS, kita dapat memisahkan konten dari presentasi, membuat kode lebih terstruktur, dan memudahkan pengelolaan tampilan situs web
